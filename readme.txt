== Description ==

Display your partners, clients or sponsors on your website in a WE - Client Logo Carousel.

**Features**

* Short code to call all logos as a Carousel [weclient_logo_slider]
* Short code to display specific category logos as a Carousel [weclient_logo_slider category="SLUG OF CAROUSEL CATEGORY"]
* You can call shortcode to php template file using below code
<pre>
 echo do_shortcode('[weclient_logo_slider]'); 
</pre>
</pre>
* Display specific category logos as a Carousel
<pre>
  echo do_shortcode('[weclient_logo_slider category="SLUG OF CAROUSEL CATEGORY"]'); 
</pre>


== Installation ==

**Installation**

1. Upload the plugin to your blog/site and activate
2. Create Partners/Clients using we-client-logo custom post type
3. Visit the WE - Client Logo Carousel Settings to update settings
4. Apply Shortcode wherever you needed

== Frequently Asked Questions ==

= How can I create WE - Client Logo Carousel? = 
After activating the plugin, you will see "WE Client Logo" tab in WordPress Admin Dashboard.

Click on "Add New Client Logo" to add logos. Just follow wp-admin->WE Client Logo->Add New Client Logo

Use "WE Client Logo Carousel Settings" tab to customise front end look, number of logos to display, carousel speed, logos order etc. 

Finally, Just insert Short code as needed on any page or post. 
= Is WE - Client Logo Carousel responsive? = 
Yes, WE - Client Logo Carousel is fully responsive, mobile friendly.

= Do I need to have coding skills to use WE - Client Logo Carousel? = 
No. You don't need any coding skill. Just use shortcode to display carousel.
